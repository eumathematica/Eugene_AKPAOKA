1️⃣ Nombres Complexes

Cours & explications

Un nombre complexe peut s'écrire z = a + ib, avec a,b réels et i²=-1. Il peut être représenté dans le plan complexe comme un point ou vecteur.

Formules & propriétés

  • Conjugué : z̄ = a - ib
  • Module : |z| = √(a²+b²)
  • Argument : θ = arg(z)
  • Forme exponentielle : z = r e^(iθ)

4️⃣ Limites et Continuité

Cours & explications

La limite d'une fonction f(x) lorsque x tend vers a permet de connaître le comportement de f(x) près de a.

Une fonction f est continue en a si : lim x→a f(x) = f(a).

Formules & propriétés

  • lim x→a [f(x) + g(x)] = lim f(x) + lim g(x)
  • lim x→a [f(x) * g(x)] = lim f(x) * lim g(x)
  • lim x→a [f(x)/g(x)] = lim f(x)/lim g(x), g(a) ≠ 0
  • lim x→∞ 1/x = 0
  • lim x→0 sin(x)/x = 1

5️⃣ Étude de Fonction

Cours & explications

L’étude de fonction consiste à analyser les variations, les extremums, les limites et le graphe d’une fonction donnée. On utilise souvent la dérivée f'(x) pour déterminer le sens de variation.

Principales étapes

  1. Déterminer le domaine de définition
  2. Calculer les limites aux bornes
  3. Calculer la dérivée f'(x)
  4. Résoudre f'(x) = 0 pour trouver les points critiques
  5. Étudier le signe de f'(x) pour obtenir le tableau de variation
  6. Tracer le graphique

Formules utiles

  • f'(x) > 0 ⇒ f croissante
  • f'(x) < 0 ⇒ f décroissante
  • f'(x) = 0 ⇒ extremum possible
  • Extremum local = minimum ou maximum sur un intervalle

Exercices

  1. Étudier les variations de f(x)=x²
  2. Étudier f(x)=x³-3x
  3. Déterminer les extremums de f(x) = -x²+4x+1
  4. Tracer le tableau de variation de f(x) = 1/x
  5. Étudier f(x)=√x sur [0,4]
  6. Déterminer les extremums de f(x) = x⁴-4x²
  7. Étudier f(x) = e^x - 2x
  8. Tracer le tableau de variation de f(x) = ln(x)
  9. Étudier f(x) = sin(x) sur [0,2π]
  10. Déterminer les extremums de f(x) = cos(x) + x
  1. Domaine ℝ, f'(x)=2x : décroissante sur (-∞,0), croissante sur (0,+∞), min en 0
  2. f'(x)=3x²-3=3(x-1)(x+1) : max local en -1, min local en 1
  3. f'(x)=-2x+4=0 → x=2 : max local f(2)=5
  4. Domaine ℝ\{0}, décroissante sur (-∞,0), décroissante sur (0,+∞)
  5. Croissante sur [0,4], min en 0, max en 4
  6. f'(x)=4x³-8x=4x(x²-2) : min locaux en ±√2, max en 0
  7. f'(x)=e^x - 2 = 0 → e^x=2 → x=ln(2)
  8. Croissante sur (0,+∞)
  9. f'(x)=cos(x) : max en 0, min en π
  10. f'(x)=-sin(x)+1=0 → sin(x)=1 → x=π/2

6️⃣ Dérivation

Cours & explications

La dérivation est un outil fondamental pour étudier les variations d’une fonction. La dérivée f'(x) mesure la pente de la tangente à la courbe de f en un point x. Elle permet de déterminer la croissance, la décroissance et les extremums.

Règles de dérivation

  • (k)' = 0
  • (xⁿ)' = n·xⁿ⁻¹
  • (u+v)' = u' + v'
  • (u·v)' = u'·v + u·v'
  • (u/v)' = (u'·v - u·v') / v²
  • (u∘v)' = (u'∘v) · v'
  • (e^u)' = u'·e^u
  • (ln u)' = u'/u
  • (sin u)' = u'·cos u
  • (cos u)' = -u'·sin u

Exercices

  1. Dériver f(x) = 3x² + 5x - 7
  2. Dériver f(x) = (2x³ - x)·(x² + 1)
  3. Dériver f(x) = (x² + 1) / (x - 1)
  4. Dériver f(x) = e^(2x)
  5. Dériver f(x) = ln(x² + 3)
  6. Dériver f(x) = sin(3x)
  7. Dériver f(x) = cos(x²)
  8. Dériver f(x) = x·e^x
  9. Dériver f(x) = (x² - 1)·ln(x)
  10. Dériver f(x) = (sin x)/(cos x)
  1. f'(x) = 6x + 5
  2. f'(x) = (6x² - 1)(x² + 1) + (2x³ - x)(2x)
  3. f'(x) = [(2x)(x - 1) - (x² + 1)(1)] / (x - 1)²
  4. f'(x) = 2e^(2x)
  5. f'(x) = (2x)/(x² + 3)
  6. f'(x) = 3cos(3x)
  7. f'(x) = -2x·sin(x²)
  8. f'(x) = e^x + x·e^x
  9. f'(x) = 2x·ln(x) + (x² - 1)/x
  10. f'(x) = (cos²x + sin²x) / cos²x = 1/cos²x

7️⃣ Continuité

Définition

Une fonction f est continue en un point a si :

  • f(a) existe,
  • limx→a f(x) existe,
  • limx→a f(x) = f(a).
Si f est continue sur un intervalle I, cela signifie que la courbe représentative ne présente ni "trou", ni "saut" sur I.

Propriétés

  • La somme, le produit et le quotient (si dénominateur ≠ 0) de fonctions continues sont continues.
  • Les fonctions polynomiales et exponentielles sont continues sur ℝ.
  • La fonction ln(x) est continue sur ]0, +∞[.
  • Les fonctions trigonométriques sin et cos sont continues sur ℝ.

Exercices

  1. Vérifier la continuité de f(x) = 3x² + 2x - 1 en x = 1.
  2. Vérifier si f(x) = (x² - 1)/(x - 1) est continue en x = 1.
  3. Vérifier la continuité de f(x) = ln(x) sur son domaine.
  4. Déterminer l’ensemble de continuité de f(x) = (x² + 1)/(x - 2).
  5. Montrer que g(x) = sin(x)/x est continue en 0 par prolongement.
  1. f(1) = 4, limx→1 f(x) = 4, donc continue en x = 1.
  2. Simplification : f(x) = x + 1 pour x ≠ 1, donc limx→1 f(x) = 2 ≠ f(1) (qui est indéfini), donc pas continue en 1.
  3. ln(x) est continue sur ]0, +∞[ par définition.
  4. Continue sur ℝ \ {2} (dénominateur nul en x = 2).
  5. Prolongement : limx→0 sin(x)/x = 1, donc g(0) = 1 ⇒ continue en 0 par prolongement.

8️⃣ Asymptotes

Définition

Une asymptote est une droite que la courbe représentative d’une fonction f approche de façon infinie, sans jamais (ou presque) la toucher, lorsque x tend vers ±∞ ou vers une valeur interdite.

Types d’asymptotes

  • Asymptote verticale : lorsque limx→a⁺ f(x) = ±∞ ou limx→a⁻ f(x) = ±∞.
  • Asymptote horizontale : lorsque limx→±∞ f(x) = L ∈ ℝ.
  • Asymptote oblique : lorsque limx→±∞ (f(x) - (mx + p)) = 0 avec m ≠ 0.

Méthode pour trouver les asymptotes

  1. Déterminer le domaine de définition de f(x).
  2. Étudier les limites aux bornes du domaine (y compris ±∞).
  3. Identifier les droites d’équation x = a (verticales), y = L (horizontales) ou y = mx + p (obliques).

Exercices

  1. Trouver les asymptotes de f(x) = 1/(x - 2).
  2. Trouver les asymptotes de f(x) = (2x² - 3x + 1)/(x - 1).
  3. Trouver les asymptotes de f(x) = (x² + 1)/(x² - 4).
  4. Trouver les asymptotes de f(x) = (3x - 1)/2x.
  5. Trouver les asymptotes de f(x) = (x³ + 2)/(x² - 1).
  1. Domaine : ℝ \ {2}, limx→2 = ±∞ → Asymptote verticale x = 2. limx→±∞ f(x) = 0 → Asymptote horizontale y = 0.
  2. Domaine : ℝ \ {1}, x = 1 verticale. Division : f(x) = 2x - 1 + 0/(x - 1) → Asymptote oblique y = 2x - 1.
  3. Domaine : ℝ \ {±2}, verticales x = 2 et x = -2. limx→±∞ = 1 → horizontale y = 1.
  4. Domaine : ℝ \ {0}, limx→0 = ±∞ verticale x = 0. limx→±∞ = 3/2 horizontale y = 1.5.
  5. Domaine : ℝ \ {±1}, verticales x = ±1. Division : f(x) = x + 0/(x² - 1) → Asymptote oblique y = x.

9️⃣ Bijections

Définition

Une application \( f : A \to B \) est une bijection si elle est à la fois injective (chaque élément de B a au plus un antécédent) et surjective (chaque élément de B a au moins un antécédent). Autrement dit, chaque élément de B correspond à un unique élément de A.

Schéma explicatif

Imagine deux ensembles A et B. Une flèche partant d’un élément de A et arrivant dans B sans que deux éléments distincts de A aient la même image, et en couvrant tout B.

Propriétés

  • Une bijection admet une fonction réciproque \( f^{-1} \).
  • La composée de deux bijections est une bijection.
  • Si \( f \) est bijective, alors elle établit une correspondance parfaite entre A et B.

Vérifier qu’une fonction est bijective

  1. Montrer que \( f \) est injective : \( f(x_1) = f(x_2) \Rightarrow x_1 = x_2 \).
  2. Montrer que \( f \) est surjective : pour tout \( y \in B \), il existe \( x \in A \) tel que \( f(x) = y \).

Exemples

  1. \( f : \mathbb{R} \to \mathbb{R},\ f(x) = 3x + 2 \) → bijective car linéaire affine avec pente ≠ 0.
  2. \( f : \mathbb{R}^* \to \mathbb{R}^*,\ f(x) = \frac{1}{x} \) → bijective, réciproque : \( f^{-1}(x) = \frac{1}{x} \).
  3. \( f : \mathbb{R} \to [0,+\infty[,\ f(x) = x^2 \) → pas bijective car non injective (x et -x ont la même image).

Exercices

  1. Montrer que \( f(x) = 5x - 1 \) est bijective sur \(\mathbb{R}\).
  2. Déterminer si \( f(x) = e^x \) est bijective sur \(\mathbb{R}\) vers \(\mathbb{R}^+\).
  3. Vérifier si \( f(x) = |x| \) est bijective sur \(\mathbb{R}^+\).
  4. Trouver la réciproque de \( f(x) = 2x + 7 \).
  5. Trouver la réciproque de \( f(x) = \frac{x-3}{4} \).
  1. Injective car 5x₁ - 1 = 5x₂ - 1 ⇒ x₁ = x₂ ; surjective car tout réel y est atteint : x = (y+1)/5.
  2. Injective sur ℝ et surjective vers ℝ⁺, donc bijective.
  3. Pas injective sur ℝ car |2| = |-2|, mais injective si on restreint à ℝ⁺.
  4. Réciproque : \( f^{-1}(x) = \frac{x - 7}{2} \).
  5. Réciproque : \( f^{-1}(x) = 4x + 3 \).

🔟 Arithmétique

Définitions clés

  • Divisibilité : Un entier \( a \) est divisible par un entier \( b \) (\( b \neq 0 \)) si \( a = b \times k \) pour un certain entier \( k \). On note \( b \mid a \).
  • PGCD : Plus grand entier qui divise simultanément deux entiers.
  • PPCM : Plus petit entier commun multiple à deux entiers non nuls.
  • Nombre premier : Entier naturel \( > 1 \) qui n'a que deux diviseurs : 1 et lui-même.

Propriétés

  • Si \( a \mid b \) et \( b \mid c \) alors \( a \mid c \).
  • Relation fondamentale : \[ a \times b = \text{PGCD}(a,b) \times \text{PPCM}(a,b) \]
  • Tout entier \( n \geq 2 \) peut s'écrire comme un produit de nombres premiers (unicité de la décomposition).

Algorithme d'Euclide (PGCD)

  1. Diviser le plus grand des deux nombres par le plus petit.
  2. Remplacer le plus grand par le reste obtenu.
  3. Répéter jusqu'à obtenir un reste nul. Le dernier diviseur non nul est le PGCD.

Exemples

  1. \( 36 \mid 108 \) car \( 108 = 36 \times 3 \).
  2. PGCD(18, 24) = 6 et PPCM(18, 24) = 72.
  3. 7 est premier ; 21 ne l'est pas car \( 21 = 3 \times 7 \).

Théorème de Bézout

Si \( d = \text{PGCD}(a,b) \), alors il existe des entiers \( u \) et \( v \) tels que : \[ au + bv = d \]

Petit théorème de Fermat

Si \( p \) est premier et \( a \) un entier non divisible par \( p \), alors : \[ a^{p-1} \equiv 1 \ (\text{mod}\ p) \]

Exercices

  1. Déterminer si 91 est un nombre premier.
  2. Calculer PGCD(48, 180) et PPCM(48, 180).
  3. Décomposer 360 en facteurs premiers.
  4. Résoudre \( 35u + 21v = 7 \) en entiers.
  5. Vérifier le petit théorème de Fermat pour \( a=3 \) et \( p=7 \).
  1. 91 = 7 × 13 ⇒ pas premier.
  2. PGCD(48,180) = 12 ; PPCM = 48 × 180 / 12 = 720.
  3. 360 = 2³ × 3² × 5.
  4. PGCD(35,21) = 7 ⇒ u=2, v=-3 est une solution.
  5. 3⁶ = 729 et 729 mod 7 = 1 ⇒ vérifié.

1️⃣1️⃣ Probabilités

Vocabulaire de base

  • Expérience aléatoire : expérience dont l’issue ne peut pas être prévue avec certitude.
  • Univers : ensemble \( \Omega \) des issues possibles.
  • Événement : sous-ensemble de \( \Omega \).
  • Événement certain : \(\Omega\) ; Événement impossible : \(\varnothing\).

Probabilité d’un événement

Si les issues sont équiprobables : \[ P(A) = \frac{\text{nombre d’issues favorables}}{\text{nombre total d’issues}} \]

Propriétés

  • 0 ≤ P(A) ≤ 1
  • P(Ω) = 1 ; P(∅) = 0
  • P(Ā) = 1 − P(A)
  • P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Probabilités conditionnelles

Si P(B) > 0 : \[ P(A \mid B) = \frac{P(A \cap B)}{P(B)} \]

Indépendance

A et B sont indépendants si : \[ P(A \cap B) = P(A) \times P(B) \]

Formule des probabilités totales

Si \( (B_1, B_2, \dots, B_n) \) est une partition de Ω : \[ P(A) = \sum_{i=1}^n P(A \mid B_i) \cdot P(B_i) \]

Théorème de Bayes

\[ P(B_j \mid A) = \frac{P(A \mid B_j) \cdot P(B_j)}{\sum_{i=1}^n P(A \mid B_i) \cdot P(B_i)} \]

Exemples

  1. Lancer un dé : P(obtenir un 4) = 1/6.
  2. Pioche d’une carte rouge dans un jeu de 52 cartes : 26/52 = 1/2.
  3. Piocher un as après avoir tiré un roi et l’avoir remis : événements indépendants.

Exercices

  1. On lance deux dés : calculer P(somme = 7).
  2. On tire deux cartes sans remise : calculer P(2 as).
  3. Une urne contient 5 boules rouges et 3 bleues : P(2 rouges avec remise) ?
  4. Vérifier si tirer un roi et tirer un as (avec remise) sont indépendants.
  5. Appliquer Bayes à un test médical ayant 95 % de fiabilité.
  1. 6 cas favorables sur 36 ⇒ 1/6.
  2. \( \frac{4}{52} \times \frac{3}{51} = \frac{12}{2652} = \frac{1}{221} \).
  3. \( \frac{5}{8} \times \frac{5}{8} = \frac{25}{64} \).
  4. Indépendants car P(Roi ∩ As) = P(Roi) × P(As).
  5. Bayes appliqué : résultat dépend de la prévalence.

1️⃣2️⃣ Statistiques

Vocabulaire et notions de base

  • Population : ensemble des individus étudiés.
  • Effectif 👎 : nombre d’individus de la population.
  • Fréquence (f) : \( f = \frac{\text{effectif de la modalité}}{\text{effectif total}} \).
  • Série statistique : tableau regroupant les valeurs observées et leurs effectifs.

Mesures de tendance centrale

  • Moyenne : \( \bar{x} = \frac{\sum x_i \cdot n_i}{\sum n_i} \)
  • Médiane : valeur qui partage la population en deux parties égales.
  • Mode : valeur la plus fréquente.

Mesures de dispersion

  • Étendue : max − min.
  • Variance : \( V = \frac{\sum n_i (x_i - \bar{x})^2}{N} \)
  • Écart-type : \( \sigma = \sqrt{V} \)

Tableau de fréquences cumulées

Permet de déterminer la médiane, les quartiles et autres indicateurs de position.

Exemple

On a relevé les notes suivantes pour 10 élèves :

Moyenne : \( \bar{x} = \frac{8 \times 2 + 10 \times 3 + 12 \times 4 + 15 \times 1}{10} = \frac{16 + 30 + 48 + 15}{10} = \frac{109}{10} = 10,9 \).

Représentations graphiques

  • Histogramme
  • Diagramme en bâtons
  • Polygone de fréquences
  • Boîte à moustaches (boxplot)

📊 Équations différentielles

1. Définition

Une équation différentielle est une équation qui relie une fonction inconnue à ses dérivées. Elle sert à modéliser des phénomènes physiques, biologiques, économiques, etc.

2. Types principaux

  • Ordre 1 : implique la dérivée première \( y'(x) \).
  • Ordre 2 : implique la dérivée seconde \( y''(x) \).
  • À coefficients constants : les coefficients sont des constantes.
  • À variables séparables : on peut isoler \( y \) et \( x \) de part et d’autre.
  • Linéaires : de la forme \( y' + a(x)y = b(x) \).

3. Méthodes de résolution

  1. Variables séparables On écrit \( \frac{dy}{dx} = f(x)g(y) \), puis on intègre : \( \int \frac{1}{g(y)}\,dy = \int f(x)\,dx + C \).
  2. Équation linéaire du 1er ordre \( y' + a(x)y = b(x) \) Solution : \( y = e^{-\int a(x)dx} \left( \int b(x) e^{\int a(x)dx}dx + C \right) \).
  3. Équations différentielles homogènes du 2e ordre à coefficients constants \( ay'' + by' + cy = 0 \) On résout l’équation caractéristique \( ar^2 + br + c = 0 \) pour obtenir la solution générale.

4. Exemples classiques

  • Refroidissement de Newton : \( \frac{dT}{dt} = -k(T - T_{\text{ext}}) \)
  • Croissance exponentielle : \( \frac{dy}{dt} = ky \)
  • Oscillateur harmonique : \( y'' + \omega^2 y = 0 \)

5. Exercices

  1. Résoudre \( \frac{dy}{dx} = 3y \) avec \( y(0) = 2 \).
  2. Résoudre \( y' + 2y = 5 \) avec \( y(0) = 1 \).
  3. Résoudre \( y'' - 4y' + 4y = 0 \).
  4. Modéliser et résoudre la loi de refroidissement de Newton avec \( k = 0.1 \), \( T_{\text{ext}} = 20 \), \( T(0) = 80 \).
  1. \( y = 2e^{3x} \).
  2. Facteur intégrant : \( e^{2x} \), solution : \( y = \frac{5}{2} + Ce^{-2x} \) → avec \( y(0) = 1 \), \( C = -\frac{3}{2} \).
  3. Équation caractéristique : \( (r - 2)^2 = 0 \) → solution : \( y = (A + Bx)e^{2x} \).
  4. Solution : \( T(t) = 20 + 60e^{-0.1t} \).
